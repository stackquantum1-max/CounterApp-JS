# Counter App

A simple, interactive **counter application** built with **HTML, CSS, and JavaScript**.  
The app allows users to increment, decrement, and reset a counter, with a **phone-style container design** and responsive layout.

---

## Features

- **Increment Counter:** Increase the number by a chosen step (default 1, can be changed in code).  
- **Decrement Counter:** Decrease the number but **never below 0**.  
- **Reset Counter:** Reset the counter back to 0.  
- **Responsive Design:** Phone-like container with centered display and buttons.  
- **Dynamic Font Size (optional):** Font size adapts when numbers grow large.  
- **Clean UI:** Simple, modern button styles with hover and click effects.
